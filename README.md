# 7mGacha

Telegram бот для сбора карт одноклассников.

## Функции

- 🎴 Получение осколков по редкости
- ⚒️ Крафт карт из 6 осколков
- 🏆 Таблица лидеров
- 📊 Профиль с очками
- 🎁 Лимитированные карты (9 в месяц)

## Редкости и очки

| Редкость | Шанс | Очки |
|----------|------|------|
| Обычная | 50% | 10 |
| Редкая | 25% | 50 |
| Эпическая | 15% | 200 |
| Легендарная | 8% | 1000 |
| Мифическая | 2% | 5000 |

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

## Настройка

Отредактируйте `config.py`:
- `COOLDOWN_SECONDS` - КД между крутками
